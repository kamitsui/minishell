### How to use

#### Compile check
> ```make``` : all target
>
> ```make N``` : chose target
>
> chose N from the below
> * ```1``` : handle_operator
> * ```2``` : traverse
> * ```3``` : parse
>

#### Run check
> ```make run``` : Run all test
>
> ```make runN``` : chose target
>
> chose N from the below
> * ```1``` : handle_operator
> * ```2``` : traverse
> * ```3``` : parse
